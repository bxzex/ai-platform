# Private AI Chat Platform

A minimalist, high-performance web interface for local AI. This platform runs state-of-the-art Large Language Models (LLMs) entirely on your device using Web-LLM technology.

## Features

- **100% Private:** Your messages and data never leave your device. All processing happens locally in your browser.
- **Offline Capable:** Once the model is initialized, you can chat without an internet connection.
- **High Performance:** Optimized for low-latency responses using hardware acceleration (WebGPU/WebGL).
- **Modern Interface:** Clean, focused chat experience with dark mode and persistent history.

## Getting Started

1. Visit the [Live Demo](https://bxzex.github.io/ai-platform/)
2. Click **Initialize Model** in the header to download the local weights (one-time setup).
3. Start chatting securely.

## Built With

- **React 19** - Frontend Framework
- **Vite** - Build Tooling
- **Web-LLM** - High-performance local LLM engine
- **Lucide React** - Icon System
- **Framer Motion** - Smooth UI transitions

## Privacy & Security

This application is designed with a "local-first" philosophy. No analytics, tracking, or cloud-syncing is implemented. Your data is stored exclusively in your browser's local storage.
